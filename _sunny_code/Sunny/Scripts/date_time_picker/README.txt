jQuery Timepicker Addon
=======================

Use
---
- To use this plugin you must include jQuery and jQuery UI with datepicker and slider

<script type="text/javascript" src="data_time_picker/jquery-ui-1.8.16.custom-Stable/js/jquery-1.6.2.min.js"></script>
<script type="text/javascript" src="data_time_picker/jquery-ui-1.8.16.custom-Stable/js/jquery-ui-1.8.16.custom.min.js"></script>

<link rel="stylesheet" type="text/css" href="data_time_picker/jquery-ui-timepicker-addon.css" />
<link rel="stylesheet" type="text/css" href="data_time_picker/jquery-ui-1.8.16.custom-Stable/css/smoothness/jquery-ui-1.8.16.custom.css" />


- Include timepicker-addon script

<script type="text/javascript" src="data_time_picker/jquery-ui-timepicker-addon.js"></script>

- now use timepicker with $('#selector').datetimepicker() or $('#selector').timepicker() //should be after the element
